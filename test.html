Spec 7.1 – lagringstest

Räknaren ökar med 1 vid varje öppning/laddning och sparas i localStorage. Diagnostiken nedan visar var koden körs.

–

Export / import (steg 5)

Ingen export/import körd ännu.

Protokoll att bocka av

  1. iPhone via Filer (file://): AirDroppa filen, öppna via Filer-appen två gånger. Räknaren ska inte öka mellan öppningarna → bekräftar att file:// på iOS är flyktigt.
  2. iPhone hemskärm (https): lägg filen på GitHub Pages, "Lägg till på hemskärmen", flygplansläge på, öppna → fungerar offline? Öppna igen efter några dagar → räknaren kvar?
  3. Mac (file://): öppna i Chrome, starta om webbläsaren → räknaren kvar?
  4. Export/import: exportera via share sheet → Dropbox, importera tillbaka via filväljaren ovan.
